London Prayer Times plugin is a sidebar widget that displays prayer times in a table. The Jama'ah (congregation) times are editable.

== Description ==

Released in the summer 2015 - The London Prayer Times Wordpress plugin is an easy to install sidebar widget that displays prayer times in a table format. It provides London Unified Prayer Times (Start times) but also Jama'ah (congregation) times. The Jama'ah times are editable so you can enter the times of your local masjid.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `prayer-times` folder to the `/wp-content/plugins/` directory or download the plugin via the inbuilt Wordpress plugin library.
1. Activate the plugin through the 'Plugins' menu in the WordPress dashboard.
1. Once activated, go to Settings > Prayer Times (screenshot 1) and enter your API KEY. If you do not have an API KEY please apply for one [here](http://www.londonprayertimes.com/api/ "http://www.londonprayertimes.com/api/").
1. The Prayer Times widget will appear as an available widget and can be dragged into your sidebar (screenshot 2).
1. On the widget dropdown menu you can override the default Jama'ah times by entering your own times.

== Frequently Asked Questions ==

= There are already Jama'ah times being displayed, whos are they? =

The jama'ah times being displayed are from [Croydon ICT](http://www.croydonict.com "www.croydonict.com"). These appear by default however you can override them in the widget dropdown menu.

= Can I edit the Jama'ah times? =

Yes. In the widget dropdown menu you can override the default times.

= Can I edit the Start times? =

No. The start times are in accordance with the London Unified Prayer Timetable.

= What cities in the UK do you have prayer times for? =

At this time we only provide times for London. When editing the prayer times widget it does ask what city you would like to display however at this time the only option is London.

= Displaying the date =

By default the widget uses "D jS M Y" which outputs i.e "Thu 28th May 2015". You can edit this, here is supporting [documentation](http://php.net/manual/en/function.date.php "http://php.net/manual/en/function.date.php").
